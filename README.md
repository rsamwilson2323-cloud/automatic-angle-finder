# 🎯 Automatic Angle Finder

A real-time computer vision project that detects hand landmarks using a webcam and calculates the angle between the wrist and index finger. Built using OpenCV and cvzone, this tool provides a simple and interactive way to visualize hand orientation for gesture-based applications.

---

## 🚀 Features

- 🎥 Real-time webcam hand tracking  
- ✋ Detects hand landmarks using cvzone  
- 📐 Calculates angle between wrist and index finger  
- 🖥️ Fullscreen display for better visualization  
- 🎯 Accurate and fast angle computation  
- ⌨️ Press ENTER to exit

## ⚙️ How It Works

1. Captures video from webcam  
2. Detects hand using cvzone HandDetector  
3. Extracts wrist and index finger tip positions  
4. Draws a line between the two points  
5. Calculates angle using trigonometry  
6. Displays angle in real-time on screen  

---

## 💡 Example

- Move your hand in front of the camera  
- The system tracks your hand  
- Angle between wrist and index finger is displayed live
